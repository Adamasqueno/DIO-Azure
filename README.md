# DIO-Azure
Apenas um Read.me com algumas informações sobre o que eu vi no bootcamp de Azure

O Microsoft Azure é uma plataforma de nuvem da Microsoft que permite criar, gerenciar e usar diversos serviços pela internet de maneira simples e sucinta que vai direto ao ponto e que com ele, é possível armazenar e ler dados, processar informações e criar soluções de forma rápida e segura.
O Azure pode até parecer complicado de primeiro momento, mas ele é simples e intuitivo, mesmo para quem está começando. 

Pelo próprio Portal do Azure, dá para criar recursos com poucos cliques, usando uma interface visual e fácil de entender, além disso, o Azure oferece muitas outras funções, como bancos de dados, automação, integração de sistemas e análise de dados.
Um ponto extremamente positivo é que você pode pagar pelo o que usar, criar uma conta de estudante ou até mesmo utilizar um cartão de crédito para utilizar os serviços de forma paga mensalmente, sendo assim, isso tudo torna o Azure prático tanto para projetos pequenos quanto para soluções maiores.
